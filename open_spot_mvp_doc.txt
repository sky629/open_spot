프로젝트 "Open-Spot" MVP 기획서
1. 프로젝트 개요 (Introduction)
프로젝트명: Open-Spot

한 줄 요약: 서울시 공공데이터를 활용하여, 예비 창업자에게 AI 기반 상권 분석 리포트를 제공하는 B2B 웹 서비스.

프로젝트 비전: 데이터 기반의 객관적인 인사이트를 통해 소상공인의 성공적인 창업을 돕고, 정보 비대칭 문제를 해결한다.

2. 우리가 해결하려는 문제 (Problem Statement)
문제: 예비 창업자들은 상권 정보를 얻기 위해 많은 시간과 비용을 소모하지만, 정보가 파편화되어 있고 해석하기 어렵다.

기존 방식의 한계: 주로 부동산이나 개인적인 경험에 의존한 의사결정은 실패 위험이 높다. 전문적인 상권 분석 리포트는 비용이 매우 비싸 소상공인이 접근하기 어렵다.

우리의 솔루션: 공공데이터를 활용하여, 저렴하고 합리적인 비용으로 누구나 쉽게 이해할 수 있는 핵심 상권 분석 데이터를 제공한다.

3. 타겟 고객 (Target Audience)
초기 핵심 타겟 (Early Adopters): 서울에서 카페, 소규모 음식점, 헬스장 등 외식업/서비스업 창업을 준비하는 30-40대 예비 창업자.

확장 타겟: 프랜차이즈 본사의 점포 개발팀, 상업용 부동산 중개인 및 컨설턴트.

4. MVP 핵심 기능 정의 (Core Features)
MVP는 "가장 핵심적인 가치를 가장 빠르게 검증"하는 데 집중합니다. 따라서 아래 기능들을 필수 기능으로 정의합니다.

F-01: 상권 분석 조건 입력

사용자는 지도를 통해 분석을 원하는 위치를 클릭하거나 주소를 입력할 수 있다.

분석할 업종을 선택할 수 있다 (초기 MVP에서는 '카페'로 고정).

분석 반경을 선택할 수 있다 (예: 300m, 500m, 1km).

F-02: 상권 분석 리포트 생성 및 조회

요청된 조건에 따라 백엔드에서 데이터 분석을 수행하고, 고유 ID를 가진 리포트를 생성한다.

사용자는 생성된 리포트 페이지에서 다음과 같은 시각화된 데이터를 확인할 수 있다.

경쟁 분석: 분석 지역 내 경쟁 카페 목록과 위치를 지도 위에 시각화.

유동인구 분석: 주변 지역의 시간대별/성별/연령대별 유동인구 분포를 차트로 시각화.

매출 분석: 주변 상권의 분기별 평균 매출액 및 동종 업종 매출 추이를 차트로 시각화.

종합 점수: 위 3가지 데이터를 기반으로 계산된 A~F 등급의 '상권 매력도' 점수 표시.

F-03: 사용자 인증 (Google 소셜 로그인)

사용자는 Google 계정을 통해 서비스에 가입하고 로그인할 수 있다.

로그인한 사용자는 과거에 분석했던 리포트 목록을 조회할 수 있다.

5. MVP 범위에서 제외할 기능 (Non-Goals)
MVP를 단순하게 유지하기 위해 아래 기능들은 초기 버전에서 제외합니다.

결제 시스템: 모든 리포트 생성 및 조회를 무료로 제공한다.

다양한 업종 선택: 분석 가능 업종을 '카페' 하나로 고정한다.

리포트 PDF 다운로드 및 공유: 웹 페이지만으로 결과를 확인한다.

실시간 데이터 스트리밍: 모든 데이터는 일/주 단위로 업데이트되는 배치(Batch) 데이터를 기반으로 한다.

6. 기술 스택 (Technology Stack)
백엔드: Kotlin, Spring Boot, Spring Cloud (Gateway, Config), Spring Security

데이터 파이프라인: Python, Pandas, GeoPandras

데이터베이스: PostgreSQL + PostGIS

프론트엔드: React (TypeScript)

인프라/배포: Docker, Kubernetes (AWS EKS 또는 ECS), GitHub Actions

지도/차트: 네이버 지도 API, ECharts.js (또는 Chart.js)

7. 성공 지표 (Success Metrics)
MVP 출시 후, 우리가 올바른 방향으로 가고 있는지 판단하기 위한 기준입니다.

주요 지표: 주간 활성 사용자 수 (WAU), 리포트 생성 요청 수

검증 목표:

출시 후 1개월 내, 100명 이상의 사용자가 회원가입을 하는가?

가입한 사용자들이 평균 3회 이상의 리포트를 생성하는가?

사용자 설문조사를 통해 "제공된 리포트가 창업 결정에 도움이 되었는가?"라는 질문에 50% 이상이 긍정적으로 답변하는가?
